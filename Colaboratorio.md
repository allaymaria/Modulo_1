# Uso de Colab
Entramos a [**Google Colab**](https://colab.research.google.com/), donde creamos un nuevo "notebook", en el botón **Editar** vamos a **Configuración de notebook** donde seleccionamos:
* Tipo de entorno de ejecución: **R** (además tenemos la opción de Python 3 y Julia);
* Acelerador de hardware: **CPU** (tenemos algunas opciones gratis y otras premium);
* Visión de entorno: **Recomendada** (última versión).
  
**+Texto** sirve para agregar texto utilizando lenguaje técnico.

**+Código** sirve para agregar código de programación.

## Códigos de tablas
Los dos puntos nos indican la posición del texto, si no ponemos dos puntos se verá igual que la columna Izquierda, es lo que llamamos Default.
```
|Izquierda|Centro|Derecha|
|:---|:---:|---:|
|1|2|3|
```

|:---|:-------:|---:|
